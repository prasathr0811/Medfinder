# MedFinder 🩺

🌐 **Live Website:** [https://medfinder-m.vercel.app/](https://medfinder-m.vercel.app/)

MedFinder is a modern, real-time medicine availability search and reservation platform designed to connect customers with local pharmacy networks. Customers can search for medicine stocks, view pharmacy details with realistic local distances (50m, 100m, etc.), and reserve medicines for secure counter pick-up verified by QR codes.

## Features 🚀

### For Customers:
* **Real-time Stock Search:** Search by brand name, active composition, category, or manufacturer.
* **Geospatial Locator:** Compares pharmacy distances and displays the closest partners.
* **Instant Reservations:** Reserve medicines directly inside the modal and view verification QR codes.
* **PDF Receipt Downloads:** Download detailed reservation invoices for offline counter collection.

### For Pharmacy Owners:
* **Interactive Dashboard:** Track active orders, pending approvals, and inventory statistics.
* **Stock & Inventory Control:** Register pharmacy location details, manage medicine stocks, prices, and upload product images.

## Technologies Used 💻

* **Frontend:** React, Tailwind CSS, Vite, TanStack Query, Framer Motion, Lucide Icons
* **Backend:** Node.js, Express.js, JWT Authentication, QRCode generator, PDFKit
* **Database:** MongoDB, Mongoose ODM
